# Maintainers

This repository is intended to be maintainable by a 4-person CS3704 project team with Klein as owner.

## Roles
- **Owner**: Klein (`@kleinpanic`) — final approval, branch protection, release authority
- **Maintainers / collaborators**: feature delivery, reviews, issue ownership, documentation upkeep

## Working agreement
- work starts from an Issue
- branch from `main`
- open PRs early
- keep PRs scoped and reviewable
- CI must pass before merge
- signed commits required on protected `main`
- do not bypass PR review or force-push protected branches

## Branch naming
- `feature/<name>`
- `fix/<name>`
- `docs/<name>`
- `chore/<name>`
- `refactor/<name>`
- `test/<name>`
- `hotfix/<name>`

Install it yourself

Takes less than a minute. Works on any Canvas LMS instance.

01

Clone & load

Clone the repo, open chrome://extensions, enable developer mode, and load the extension/ folder.

02

Add your token

Generate a Canvas API token in Account → Settings → New Access Token, then paste it in the extension settings.

03

Click the badge

The toolbar badge shows upcoming assignment count. Click to open the full dashboard.

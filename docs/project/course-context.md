## Project direction
The project is a Canvas-centered productivity tool focused on deadline visibility, prioritization, and improved task interaction. Earlier course materials describe the project as a Canvas deadline/task tracker that may exist as a TUI, web app, or browser extension.

## Documented user needs from earlier milestones
- centralized dashboard for assignments
- reminders and prioritization for upcoming deadlines
- simple and intuitive UI
- reduced risk of missed deadlines
- interest in browser-extension style access for quick deadline checks
